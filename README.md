# vk-clone

**Todd Motto** [styleguide](https://github.com/toddmotto/angular-styleguide)

+ ES2015
+ webpack
+ angular 1.5.8 (components)
+ ui-router


*******
#### Dependencies:

```
bower i
npm i
```

*******

#### Run server from *public* directory:

```
python -m http.server 9000
```

*******

#### Build with webpack:

for production:

```
NODE_ENV=public webpack
```

for developer (default):

```
webpack
```
